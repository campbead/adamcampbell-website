I recently participated in the There and Back Again data visualisation competition from Stats NZ. I wanted to quickly make a post to feature the app I made, called How Dunedin Gets to Work. This project is meant to be a lightweight app to quickly visualise the modes of transportation people of Dunedin, NZ use to get to work. I used Leaflet for the mapping components and created the app using R-Shiny. Source code for this project is available on GitHub.
